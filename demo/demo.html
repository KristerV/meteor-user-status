Instructions

  • Log in with any username.
  • Log in from other computers with the same username or open multiple tabs. These will appear as sessions/connections under the same user.
  • Click or keypress events reset the idle timer. The default idle threshold is 30 minutes.
  • Idle session information is transmitted to the server and multiplexed into the user document.
  • Play with the idle timer to try different client-side settings.
